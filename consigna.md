# Proyecto Integrador – Sistemas Operativos

**Docentes:** Lic. Gustavo A. Funes & Ing. Juan Frattin  

---

## 📌 Temas

- Comandos Linux en profundidad

---

## 🎯 Objetivos

- Comprender los conceptos detrás de los comandos utilizados en sistemas operativos tipo Linux.
- Desarrollar habilidades blandas para el trabajo en equipo y exposición oral.

---

## 🧾 Enunciado

Los estudiantes deberán utilizar los comandos aprendidos en las diferentes guías prácticas para crear automatizaciones mediante **shell scripting en Linux**.

Se conformarán grupos, cada uno responsable de un proyecto diferente.

Cada proyecto deberá implementarse como:

- Script **Bash interactivo**
- Menú de opciones
- Manejo de errores
- Código comentado

---

## ⚙️ Instrucciones

### 🐧 Plataforma
- Linux (Ubuntu, Debian, Fedora, etc.)
- ❌ No se aceptan scripts exclusivos de macOS o Windows

### 💻 Código en texto plano
- El código fuente **DEBE estar copiado como texto**
- ❌ No se aceptan capturas como única evidencia

### 📝 Comentarios
- Cada sección lógica debe incluir explicación
- Variables en **MAYÚSCULAS y descriptivas**
